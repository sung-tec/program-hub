이환 디스코드 조합·발동조건·운용법

디스코드는 암·주·혼 속성을 엮어 보스의 브레이크를 빠르게 당기는 3속성 에스퍼 사이클입니다. 복잡한 속성표보다 바로 조합을 짤 수 있게 발동 조건, 추천 조합, 티어, 운용 순서를 한 화면 기준으로 정리했습니다.

기준일: 2026-05-08 한국시간 핵심: 암 + 주 + 혼 용도: 보스 브레이크 / 패리 폭딜
핵심 요약 발동조건 조합 티어 운용법 실패 체크 출처

한 줄 결론

디스코드는 스코치 + 노바가 같은 대상에게 겹칠 때 터지는 브레이크 특화 반응입니다.

필수 캐릭터

현재는 암속성 축 때문에 다포딜이 사실상 고정입니다.

추천 시작 조합

다포딜 / 사키리 / 나나리 / 하니엘이 가장 무난합니다.

표기 기준: 국내 표기는 암·주·혼, 다포딜, 사키리, 하니엘, 백장, 나나리, 구원, 우미츠키를 우선 사용했습니다. 해외 자료의 Chaos/Incantation/Psyche는 각각 암/주/혼으로 옮겼습니다. 일부 해외 팀 페이지는 디스코드 설명에 오타성 표기가 섞여 있어, 발동조건은 인벤의 속성 구조와 Prydwen·AllClash의 스코치+노바 설명을 기준으로 정리했습니다.

핵심 요약

무엇이 좋은가

디스코드는 체력 자체를 녹이는 반응이라기보다 브레이크 게이지를 크게 압박하는 반응입니다. 보스가 빨리 브레이크되면 이후 궁극기와 폭딜을 안전하게 넣을 수 있습니다.

언제 쓰나

잡몹보다는 보스, 주간 보스, 궤도 너머, 브레이크 창이 중요한 전투에서 가치가 큽니다. 잡몹은 차지/블라썸처럼 단순하고 넓은 조합이 더 편할 수 있습니다.

가장 쉬운 기억법

다포딜 + 주속성 1명 + 혼속성 1명이면 디스코드 준비가 됩니다. 4번째 자리는 메인 딜러, 버퍼, 생존 보강 중 부족한 것을 넣습니다.

발동조건

디스코드는 3속성 반응이지만, 전투에서는 먼저 2속성 반응을 만들고 그 위에 다른 2속성 반응을 겹치는 느낌으로 보면 쉽습니다.

조건 쉬운 설명 추천 캐릭터 주의점
암속성 스코치와 노바 양쪽에 모두 끼는 중심축입니다. 다포딜 현재 공개 1.0 기준 디스코드는 다포딜 의존도가 매우 높습니다.
주속성 암속성과 만나 스코치를 만듭니다. 사키리, 백장, 아들러 사키리는 버프/제어, 백장은 고점 딜, 아들러는 생존과 디버프 쪽입니다.
혼속성 암속성과 만나 노바를 만듭니다. 하니엘, 파디아, 우미츠키 하니엘은 버프, 파디아는 생존, 우미츠키는 공격형 대체입니다.
같은 대상 스코치와 노바가 같은 적에게 겹쳐야 디스코드가 의미 있게 터집니다. 보스전 권장 여러 잡몹에게 흩어지면 체감이 크게 떨어집니다.

조합 티어표

티어는 공개 공략, 현재 캐릭터 평가, 발동 안정성, 조작 난이도를 같이 본 실전 추천도입니다. 보유 캐릭터와 각성 상태에 따라 순위는 달라질 수 있습니다.

각 조합별 운용방식

다포딜 / 사키리 / 나나리 / 하니엘

추천

나나리가 평소 딜을 담당하고, 디스코드 타이밍에는 사키리·다포딜·하니엘 축으로 반응을 터뜨립니다.

  1. 하니엘 스킬/궁으로 버프를 깔고 시작합니다.
  2. 나나리로 필드 딜을 하며 사이클 게이지를 채웁니다.
  3. 사키리와 다포딜로 스코치를 먼저 준비합니다.
  4. 하니엘과 다포딜로 노바를 겹쳐 디스코드를 터뜨립니다.
  5. 브레이크 직전이나 패리 타이밍에 다포딜로 들어가 폭딜을 넣습니다.

백장 / 다포딜 / 사키리 / 하니엘

고점

백장과 사키리의 스코치 축을 강하게 쓰는 조합입니다. 손이 바쁘지만 보스전 고점이 좋습니다.

  1. 하니엘 버프를 깔고 사키리로 제어/버프를 준비합니다.
  2. 백장으로 필드 딜을 하며 스코치 기반 피해를 누적합니다.
  3. 다포딜을 교대/패리 타이밍에 넣어 암속성 축을 유지합니다.
  4. 스코치가 붙은 대상에게 하니엘-다포딜 노바를 겹쳐 디스코드를 터뜨립니다.
  5. 브레이크 후 백장과 다포딜의 강한 스킬을 몰아칩니다.

다포딜 / 하니엘 / 아들러 / 구원

안정

백장·사키리 없이 디스코드를 굴리는 접근성 조합입니다. 아들러가 보호막과 주속성 자리를 동시에 봅니다.

  1. 구원으로 몹을 모으거나 보스 주변을 정리합니다.
  2. 아들러로 보호막을 켜고 주속성 반응을 준비합니다.
  3. 하니엘 스킬로 공격 버프와 노바 축을 준비합니다.
  4. 다포딜을 패리/교대 타이밍에 넣어 스코치와 노바를 이어 줍니다.
  5. 디스코드로 브레이크를 앞당긴 뒤 구원 또는 다포딜로 마무리합니다.

백장 / 다포딜 / 하니엘 / 파디아

생존

생존을 크게 올린 보스전 조합입니다. 사키리가 없어서 최고 딜은 낮지만 안정성이 좋습니다.

  1. 파디아로 위험 패턴 대응과 생존을 보강합니다.
  2. 하니엘 버프 후 백장으로 딜을 이어 갑니다.
  3. 다포딜로 스코치/노바 연결을 유지합니다.
  4. 디스코드는 무리해서 첫 사이클에 터뜨리기보다 패리 후 게이지가 찼을 때 노립니다.
  5. 브레이크 창에는 백장 딜을 우선하고, 다포딜은 패리/폭딜 보조로 씁니다.

발동이 안 될 때 체크

스코치를 먼저 붙이기

스코치는 지속시간이 길고, 노바는 짧게 터지는 쪽입니다. 그래서 보통 스코치 → 노바 → 디스코드 순서가 더 쉽습니다.

같은 적을 때리기

여러 적에게 반응이 흩어지면 디스코드가 잘 안 보입니다. 보스나 큰 적 하나를 고정하고 같은 대상에게 스코치와 노바를 겹치세요.

패리로 게이지 채우기

디스코드는 사이클 게이지가 늦게 차면 실패하기 쉽습니다. 보스의 패리 가능 공격에 맞춰 교대/패리를 성공시키면 다음 반응을 훨씬 빨리 만들 수 있습니다.

첫 사이클 집착 금지

전투 시작하자마자 무조건 디스코드를 터뜨릴 필요는 없습니다. 버프를 깔고 게이지를 준비한 뒤, 브레이크를 앞당기고 싶을 때 넣는 편이 안정적입니다.
